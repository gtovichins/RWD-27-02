# Responsive Web Development - Projeto Jiboia Amazônica

## Contexto Acadêmico
Este projeto foi desenvolvido como parte da disciplina de *Responsive Web Development* da UNIVALI. A incumbência consistiu na criação de uma interface web estática utilizando HTML, focada na apresentação estruturada de informações sobre um animal específico. Complementarmente, utilizei CSS para a criação de uma interface elaborada, agradável e totalmente adaptável a diferentes dispositivos.

## Objetivo
O objetivo principal foi aplicar conceitos fundamentais de desenvolvimento web, tais como:
* **Semântica HTML:** Uso correto de tags para melhor acessibilidade e SEO.
* **Layout Responsivo:** Adaptação da interface para diferentes resoluções de tela.
* **Estilização Avançada:** Implementação de camadas de design, tipografia e tratamento de imagens.
